# NASA Meteorite Tracking Database

A Java application designed to parse, filter, and analyze NASA meteorite landing data. This project demonstrates intermediate object-oriented design patterns, JSON data handling, and an interactive user interface.

## 🚀 Features
* **JSON Data Parsing:** Dynamically reads and loads complex geographic and historical meteorite records from an external data source using Gson.
* **Interactive UI:** A custom terminal-based user interface (`UserInterface.java`) that allows users to navigate, filter, and search through database records efficiently.
* **Geolocation & Data Modeling:** Features robust class structures like `Meteorite.java` and `Geolocation.java` to handle coordinate and tracking logic cleanly.

## 🛠️ Technologies Used
* **Language:** Java
* **Libraries:** Google Gson (for JSON parsing)
* **Concepts:** File I/O, Data Modeling, Encapsulation, Text-based UI Design

## 📦 How to Run the Project
1. Clone the repository.
2. Ensure you have the Java Development Kit (JDK) and the included Gson `.jar` library in your classpath.
3. Compile and run `Main.java`.
